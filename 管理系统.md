# 管理系统

## 系统概述

管理系统是 ClassNet 的后台管理模块，为管理员提供用户管理、系统监控和资源管理等功能。

## 访问权限

- **访问限制**: 仅管理员账户可访问
- **权限验证**: 需要验证管理员身份
- **入口位置**: 通过设置页 → 系统管理进入

## 核心功能

### 1. 用户账户管理

#### 功能列表

##### 1.1 查看用户列表
- 显示所有已注册用户
- 支持搜索和筛选
- 显示用户基本信息

**显示信息**:
- 用户 ID
- 自定义网名
- 真实姓名
- 性别
- 注册时间
- 账户状态（正常/禁用）
- 最后登录时间

##### 1.2 禁用/启用账户
- 管理员可禁用违规用户账户
- 支持批量禁用
- 禁用后用户无法登录

**操作流程**:
```
1. 在用户列表中选择目标用户
   ↓
2. 点击"禁用账户"按钮
   ↓
3. 填写禁用原因（可选）
   ↓
4. 确认操作
   ↓
5. 用户状态更新为"禁用"
```

##### 1.3 删除账户
- 永久删除用户账户数据
- 支持批量删除
- 删除前需确认

**注意事项**:
- 删除操作不可恢复
- 删除用户时同时删除相关数据（聊天记录等）
- 需要二次确认防止误操作

##### 1.4 发送广播
- 向所有用户发送系统广播
- 广播内容显示在超能岛
- 支持广播历史管理

**广播内容**:
- 系统通知
- 重要公告
- 活动信息
- 紧急通知

**操作界面**:

## 技术实现要点

### 1. 权限验证

#### 管理员验证中间件
```javascript
// 伪代码示例
function adminMiddleware(req, res, next) {
  // 检查是否登录
  if (!req.session || !req.session.userId) {
    return res.redirect('/login');
  }
  
  // 检查是否为管理员
  if (!req.session.isAdmin) {
    return res.status(403).json({ error: '无权限访问' });
  }
  
  next();
}
```

// 获取用户列表
router.get('/api/admin/users', async (req, res) => {

### 3. 文件管理 API

```javascript
// 文件操作 API
const fs = require('fs').promises;
const path = require('path');

// 删除文件
router.delete('/api/admin/resources', async (req, res) => {
  const { filePath } = req.body;
  const fullPath = path.join(RESOURCES_DIR, filePath);
  
  // 安全检查
  if (!fullPath.startsWith(RESOURCES_DIR)) {
    return res.status(403).json({ error: '禁止访问' });
  }
  
  await fs.unlink(fullPath);
  res.json({ success: true });
});

// 重命名文件
router.patch('/api/admin/resources/rename', async (req, res) => {
  const { oldPath, newPath } = req.body;
  
  const oldFullPath = path.join(RESOURCES_DIR, oldPath);
  const newFullPath = path.join(RESOURCES_DIR, newPath);
  
  // 安全检查
  if (!oldFullPath.startsWith(RESOURCES_DIR) || 
      !newFullPath.startsWith(RESOURCES_DIR)) {
    return res.status(403).json({ error: '禁止访问' });
  }
  
  await fs.rename(oldFullPath, newFullPath);
  res.json({ success: true });
});

// 移动文件
router.post('/api/admin/resources/move', async (req, res) => {
  const { sourcePaths, targetDir } = req.body;
  
  // 批量移动文件
  for (const sourcePath of sourcePaths) {
    const sourceFullPath = path.join(RESOURCES_DIR, sourcePath);
    const targetFullPath = path.join(RESOURCES_DIR, targetDir, path.basename(sourcePath));
    
    await fs.rename(sourceFullPath, targetFullPath);
  }
  
  res.json({ success: true });
});
```
