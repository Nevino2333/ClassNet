# 天气系统

## 系统概述

天气系统是 ClassNet 的天气信息展示模块，通过调用和风天气 API 获取实时天气数据和预报信息。

## 数据源

### 和风天气 API

#### API 文档

- **官方文档**: [和风天气开发服务](https://dev.qweather.com/docs/api/)
- **API 类型**: JWT
- **数据格式**: JSON

#### 核心接口

##### 1. 实时天气

```
GET https://devapi.qweather.com/v7/weather/now
参数:
  - location: 地点 ID（城市编码）
  - key: API 密钥
```

##### 2. 逐小时天气预报

```
GET https://devapi.qweather.com/v7/weather/24h
参数:
  - location: 地点 ID
  - key: API 密钥
```

##### 3. 七天天气预报

```
GET https://devapi.qweather.com/v7/weather/7d
参数:
  - location: 地点 ID
  - key: API 密钥
```

##### 4. 城市搜索

```
GET https://geoapi.qweather.com/v2/city/lookup
参数:
  - location: 城市名称
  - key: API 密钥
```

## 核心功能

### 1. 当前天气信息

#### 显示内容

| 信息项  | 说明      | 数据来源                          |
| ---- | ------- | ----------------------------- |
| 温度   | 当前温度（℃） | now\.temp                     |
| 天气状况 | 晴/雨/多云等 | now\.text                     |
| 天气图标 | 对应天气图标  | now\.icon                     |
| 体感温度 | 体感温度    | now\.feelsLike                |
| 相对湿度 | 湿度百分比   | now\.humidity                 |
| 风向风力 | 风向和风力等级 | now\.windDir + now\.windScale |

#### 界面示例

```
┌─────────────────────────────────┐
│        🌤️ 多云                   │
│        25°C                     │
│        体感 27°C                │
│                                 │
│   湿度：65%  东南风 3 级          │
└─────────────────────────────────┘
```

## 技术实现要点

### 1. API 调用封装

#### 服务端中转（必须）

根据开发规范，所有 API 调用必须经过服务器中转。

```javascript
// 服务端 API 封装示例
const axios = require('axios');

const QWEATHER_KEY = process.env.QWEATHER_KEY;
const BASE_URL = 'https://devapi.qweather.com/v7';

async function fetchWeather(endpoint, params) {
  const response = await axios.get(`${BASE_URL}/${endpoint}`, {
    params: {
      key: QWEATHER_KEY,
      ...params
    }
  });
  return response.data;
}

// 获取实时天气
async function getCurrentWeather(location) {
  const data = await fetchWeather('weather/now', { location });
  return data.now;
}

// 获取逐小时预报
async function getHourlyForecast(location) {
  const data = await fetchWeather('weather/24h', { location });
  return data.hourly;
}

// 获取七天预报
async function getDailyForecast(location) {
  const data = await fetchWeather('weather/7d', { location });
  return data.daily;
}
```

### 2. 前端调用

```javascript
// 前端调用示例
async function loadWeatherData() {
  try {
    // 获取实时天气
    const current = await fetch('/api/weather/current');
    
    // 获取逐小时预报
    const hourly = await fetch('/api/weather/hourly');
    
    // 获取七天预报
    const daily = await fetch('/api/weather/daily');
    
    updateUI(current, hourly, daily);
  } catch (error) {
    console.error('获取天气数据失败:', error);
    showError('天气数据加载失败');
  }
}
```

### 3. 数据缓存

#### 缓存策略

```javascript
// 缓存配置
const CACHE_CONFIG = {
  current: 10 * 60 * 1000,    // 实时天气缓存 10 分钟
  hourly: 30 * 60 * 1000,     // 逐小时预报缓存 30 分钟
  daily: 60 * 60 * 1000       // 七天预报缓存 1 小时
};

// 缓存实现
const cache = new Map();

function getCachedData(key, fetchFn, ttl) {
  const cached = cache.get(key);
  if (cached && Date.now() - cached.timestamp < ttl) {
    return cached.data;
  }
  
  const data = fetchFn();
  cache.set(key, {
    data,
    timestamp: Date.now()
  });
  return data;
}
```

## 错误处理

### 1. API 错误

- API 密钥无效
- 请求次数超限
- 网络错误

### 2. 降级方案

```javascript
// 错误处理示例
function handleWeatherError(error) {
  if (error.code === 'API_LIMIT') {
    showCachedData();  // 显示缓存数据
  } else if (error.code === 'NETWORK_ERROR') {
    showOfflineMode(); // 显示离线模式
  }
}
```
