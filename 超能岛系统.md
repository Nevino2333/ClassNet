# 超能岛系统

## 系统概述

超能岛系统是 ClassNet 的顶部信息展示组件，参考 Apple 公司的灵动岛（Dynamic Island）设计，强调实用性与美观性的结合，提供流畅的弹性动画效果。

## 设计理念

- **参考设计**: Apple Dynamic Island（灵动岛）
- **设计原则**: 实用优先、美观精致、动画流畅
- **位置**: 桌面/应用顶部中央区域

## 核心功能

### 1. 空闲状态 - 广播显示

#### 功能描述
当超能岛处于空闲状态（无其他信息显示）时，默认显示管理系统发布的广播信息。

#### 广播内容
- 系统通知
- 重要公告
- 活动信息
- 紧急通知

#### 显示形式
```
┌─────────────────────────────────┐
│ 📢 欢迎使用 ClassNet 系统        │
└─────────────────────────────────┘
```

#### 技术要求
- 支持广播内容滚动播放（多条广播时）
- 支持富文本格式（emoji、简单样式）
- 自动轮播间隔可配置（建议 5-10 秒）

### 2. 聊天系统提醒

#### 触发场景
- 收到新消息通知
- 收到@提及
- 收到系统消息

#### 显示形式
```
┌─────────────────────────────────┐
│ 💬 张三：你好，在吗？              │
└─────────────────────────────────┘
```

#### 交互行为
- 点击可跳转至聊天系统
- 显示后自动收缩回空闲状态
- 支持多条消息队列显示

### 3. 其他系统通知（扩展）

#### 潜在应用场景
| 系统 | 通知类型 | 示例 |
|------|---------|------|
| 天气系统 | 天气预警 | 暴雨预警、高温提醒 |
| AI 聊天 | 回复通知 | AI 生成完成提醒 |
| 资源仓库 | 下载完成 | 资源加载完成通知 |

## 状态管理

### 状态流转图
```
┌─────────────┐
│   空闲状态   │◄────────────────────┐
│ (显示广播)   │                     │
└──────┬──────┘                     │
       │ 收到通知                    │ 通知显示完毕
       ▼                            │
┌─────────────┐                     │
│   通知状态   │─────────────────────┘
│ (显示提醒)   │
└─────────────┘
```

### 优先级队列
```
1. 紧急系统通知（最高优先级）
2. 聊天消息提醒
3. 应用状态通知
4. 广播信息（默认状态）
```

## 技术实现要点

### 1. 动画效果

#### 弹性动画
- **展开动画**: 收到通知时平滑展开
- **收缩动画**: 通知完毕后弹性收缩
- **物理效果**: 模拟弹簧物理效果，增强真实感

#### 动画参数建议
```javascript
// 伪代码示例
const animationConfig = {
  duration: 300,        // 动画时长（毫秒）
  easing: 'spring',     // 弹性缓动
  stiffness: 400,       // 弹簧刚度
  damping: 15           // 阻尼系数
};
```

### 2. 组件结构
```
SuperIsland/
├── Container          # 主容器
├── Content            # 内容显示区域
│   ├── Broadcast      # 广播内容
│   ├── ChatNotify     # 聊天通知
│   └── SystemNotify   # 系统通知
└── AnimationController # 动画控制器
```

### 3. 数据通信

#### 与管理系统通信
```javascript
// 获取广播信息
GET /api/admin/broadcasts

// 响应示例
{
  broadcasts: [
    { id: 1, content: "欢迎使用 ClassNet", priority: "normal" }
  ]
}
```

#### 与聊天系统通信
```javascript
// 监听聊天消息（WebSocket）
ws.on('message', (data) => {
  superIsland.showNotify({
    type: 'chat',
    sender: data.sender,
    content: data.content
  });
});
```

## 与其他系统的关系

| 关联系统 | 关联方式 | 说明 |
|---------|---------|------|
| 管理系统 | 数据获取 | 获取广播信息 |
| 聊天系统 | 消息监听 | 接收聊天消息通知 |
| 天气系统 | 预警接收 | 接收天气预警信息（扩展） |
| 所有系统 | 通知展示 | 作为统一的通知展示入口 |

## 用户体验要求

### 1. 视觉设计
- 使用毛玻璃效果（backdrop-filter）
- 圆角设计，符合 iOS 风格
- 深色背景，确保内容清晰可见
- 适配亮色/暗色模式

### 2. 交互体验
- 通知自动消失（3-5 秒）
- 点击通知可跳转至对应应用
- 支持手势操作（如滑动关闭）
- 不影响用户当前操作

### 3. 性能要求
- 动画帧率保持 60fps
- 通知响应延迟 < 100ms
- 避免频繁闪烁和抖动
