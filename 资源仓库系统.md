# 资源仓库系统

## 系统概述

资源仓库系统是 ClassNet 的文件资源管理与预览模块，主要用于用户查看和预览各类资源文件，不支持文件上传功能。

## 仓库位置

- **物理路径**: `D:\Move\Documents\AI_projects\ClassNet\Resources`
- **访问方式**: 通过 Web 界面访问
- **权限控制**: 所有已登录用户可访问

## 核心功能

## 技术实现要点

### 1. 文件服务

#### 服务端文件访问
```javascript
// Node.js 文件服务示例
const express = require('express');
const path = require('path');
const fs = require('fs');

const router = express.Router();
const RESOURCES_DIR = path.resolve(__dirname, '../Resources');

// 获取文件列表
router.get('/api/resources', (req, res) => {
  const dirPath = req.query.path || '';
  const fullPath = path.join(RESOURCES_DIR, dirPath);
  
  // 安全检查：防止目录遍历攻击
  if (!fullPath.startsWith(RESOURCES_DIR)) {
    return res.status(403).json({ error: '禁止访问' });
  }
  
  const files = fs.readdirSync(fullPath);
  const fileList = files.map(file => {
    const stat = fs.statSync(path.join(fullPath, file));
    return {
      name: file,
      type: stat.isDirectory() ? 'folder' : 'file',
      size: stat.size,
      modified: stat.mtime
    };
  });
  
  res.json({ files: fileList });
});

// 文件下载
router.get('/api/resources/download', (req, res) => {
  const filePath = req.query.path;
  const fullPath = path.join(RESOURCES_DIR, filePath);
  
  // 安全检查
  if (!fullPath.startsWith(RESOURCES_DIR)) {
    return res.status(403).send('禁止访问');
  }
  
  res.download(fullPath);
});
```

### 2. 前端组件

#### 文件列表组件
```javascript
// 伪代码示例
function FileList({ path, onFileClick }) {
  const [files, setFiles] = useState([]);
  
  useEffect(() => {
    fetch(`/api/resources?path=${path}`)
      .then(res => res.json())
      .then(data => setFiles(data.files));
  }, [path]);
  
  return (
    <div className="file-list">
      {files.map(file => (
        <FileItem 
          key={file.name}
          file={file}
          onClick={() => onFileClick(file)}
        />
      ))}
    </div>
  );
}
```

### 3. 安全考虑

#### 目录遍历防护
```javascript
// 防止目录遍历攻击
function sanitizePath(userPath) {
  const safePath = path.normalize(userPath).replace(/^(\.\.(\/|\\|$))+/, '');
  return safePath;
}

// 验证路径
function isPathSafe(userPath, baseDir) {
  const fullPath = path.resolve(baseDir, userPath);
  return fullPath.startsWith(baseDir);
}
```
