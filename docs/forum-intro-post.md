# ClassNet —— 自建智慧校园平台，班级的数字化中枢

> **一句话：** 一个班级里能用到的一切——聊天、AI 助手、社区论坛、云盘、笔记、天气、音乐——全在一个网页里。

---

## 为什么做这个？

我是班级的技术爱好者。学校有各种平台，但要么功能割裂（聊天一个 App、文件又一个、AI 又要翻墙），要么数据不在自己手里。

所以我从零搭建了 **ClassNet**——跑在自己的机器上，班里同学浏览器打开就能用。前后写了两个多月，AI 辅助 coding，一个人全栈。

---

## 能干什么？

### 💬 即时通讯
- WebSocket 实时聊天，支持私聊 + 群聊 + 广播
- @提及、消息引用回复、表情反应、转发
- 群聊成员管理（踢人/禁言/转让群主）

### 🤖 AI 对话助手
- 接入云端大模型，支持 Markdown 回复
- LaTeX 数学公式渲染（完整文档 + KaTeX 行内公式）
- Mermaid 流程图/时序图/甘特图
- 对话记录可转发到论坛或聊天

### 📝 社区论坛
- 发帖/评论/点赞/转发
- Markdown + LaTeX + Mermaid 编辑器
- 匿名发帖模式
- 等级系统（每日签到、经验值、连续签到奖励）

### ☁️ 云盘
- 文件上传/下载/预览（图片/视频/音频）
- 笔记编辑器（Markdown + LaTeX + Mermaid）
- 免登录上传码（访客通过上传码提交文件）

### 📋 笔记系统
- 专业 Markdown 编辑器（**粗体**/*斜体*/~删除线~/`代码`/表格/任务列表）
- LaTeX 完整文档渲染 + KaTeX 数学公式
- Mermaid 图表（流程图/时序图/甘特图/类图/ER 图/饼图……共 20+ 种）
- 文件附件管理
- 支持全屏专注模式
- 导入/导出 .md 文件
- 笔记可设为仅自己可见
- 云端实时保存

### 🎵 音乐盒子
- 内置播放器（播放/暂停/上一曲/下一曲）
- 歌单管理（创建/编辑/删除）
- 支持本地音乐文件上传
- 歌单可转发到论坛

### 🌤️ 天气
- 实时天气卡片（温度/湿度/风力/能见度）
- 7 日天气预报
- 生活指数（紫外线/穿衣/运动等）

### ⚙️ 系统功能
- 深色/浅色主题 + 动态壁纸（内置渐变 + 自定义图片/视频）
- 等级系统（签到/发帖/评论/获赞 赚经验）
- 管理面板（用户管理/广播通知/应用管控/资源管理）
- 版本号自动管理（SemVer + 构建时自动递增）
- 新版本桌面角标提示 + 更新介绍

## 技术栈

| 层 | 技术 |
|----|------|
| 前端 | Vue 2.7 + Vue Router + Vuex |
| 构建 | Vite 5 |
| 样式 | SCSS + CSS 变量 + iOS 风格设计系统 |
| 后端 | Node.js + Express |
| 数据库 | SQLite（better-sqlite3，WAL 模式） |
| 实时通信 | WebSocket（ws） |
| 图表 | Mermaid 11 |
| 数学 | KaTeX 0.16 |
| Markdown | marked 4 |
| 视频 | Video.js 8 + HLS |
| 认证 | bcryptjs + JWT（jose） |

## 架构一览

```
浏览器 ──→ Vue 2 SPA（Vite 构建）
              │
         ┌────┴────┐
         │  HTTP    │  WebSocket
         ▼          ▼
      Express    ws 服务
      (15 个路由)  (聊天 + 中继)
         │
         ▼
      SQLite（WAL）
```

单进程全包，`node server/src/app.js` 一条命令启动。

## 开发心得

### 1. AI 写代码 ≠ 不管代码
这个项目 80% 的代码是 Claude 写的。但 AI 写的代码需要你来——审查逻辑、统一风格、修边界 bug。AI 是"写得快"，你是"写得对"。

### 2. 版本号省心方案
SemVer 的 PATCH 号构建时自动 +1，日常完全不碰版本号。新功能完成时手动改一下 MINOR。changelog 从 git log 自动生成。

### 3. 单文件 SQLite 真香
班级场景 50 人以内，SQLite 完全够用。零配置、备份就是一个文件复制。
