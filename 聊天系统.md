# 聊天系统

## 系统概述

聊天系统是 ClassNet 的即时通讯模块，参考 QQ iPad 版本设计，为用户提供公共聊天室、群聊和私聊功能。

## 设计参考

- **参考产品**: QQ iPad 版
- **设计风格**: 简洁、直观、易用的平板优化界面
- **设备适配**: 横屏平板设备

## 核心功能

### 1. 公共聊天室

#### 功能描述

所有已注册用户均可进入的公共聊天空间，支持多人同时在线聊天。

#### 特性

- 所有注册用户默认可加入
- 支持实时消息显示
- 显示在线用户列表
- 支持消息历史记录

#### 界面布局

### 2. 群聊功能

#### 创建群聊

- 用户可创建自己的群聊
- 邀请其他用户加入群聊
- 自定义群聊名称

#### 群聊管理

- 群主可管理群成员
- 支持退出群聊
- 显示群聊成员列表

#### 群聊界面

```
┌──────────────────────────────────────────────────────┐
│  群聊名称                                [成员列表]   │

├──────────────────────────────────────────────────────┤
│  [输入框]                              [发送]  [表情] │
└──────────────────────────────────────────────────────┘
```

### 3. 私聊功能

#### 功能描述

用户之间的一对一私密聊天。

#### 特性

- 从用户列表选择聊天对象
- 显示聊天历史记录
- 显示对方在线状态

#### 发起私聊

1. 在公共聊天室或用户列表中选择用户
2. 点击"私聊"按钮
3. 进入私聊界面

### 4. 消息类型支持

#### 支持的消息类型

| 类型       | 支持 | 说明          |
| -------- | -- | ----------- |
| 文本消息     | ✅  | 支持普通文本      |
| Emoji 表情 | ✅  | 支持表情符号选择和发送 |
| 文件消息     | ❌  | 不支持发送文件     |
| 语音消息     | ❌  | 不支持发送语音     |
| 图片消息     | ❌  | 不支持发送图片     |

#### Emoji 表情功能

- 提供常用表情面板
- 支持 emoji 符号直接显示
- 表情选择器分类展示

## 技术实现要点

### 1. 实时通信

#### WebSocket 连接

```javascript
// 建立 WebSocket 连接
const ws = new WebSocket('ws://server/chat');

// 监听消息
ws.onmessage = (event) => {
  const message = JSON.parse(event.data);
  displayMessage(message);
};

// 发送消息
function sendMessage(content) {
  ws.send(JSON.stringify({
    type: 'text',
    content: content,
    timestamp: Date.now()
  }));
}
```

#### 消息格式

```javascript
{
  id: "消息唯一 ID",
  type: "text|emoji",
  content: "消息内容",
  sender: {
    id: "用户 ID",
    username: "用户名"
  },
  timestamp: "时间戳",
  roomId: "聊天室/群聊 ID"
}
```

### 2. 数据存储

#### 消息历史记录

- 使用数据库存储聊天记录
- 支持按会话查询历史消息
- 分页加载历史消息

#### 数据结构示例

```javascript
// MongoDB 示例
{
  _id: ObjectId,
  roomId: String,      // 聊天室/群聊 ID
  senderId: String,    // 发送者 ID
  content: String,     // 消息内容
  type: String,        // 消息类型
  createdAt: Date,     // 发送时间
  read: Boolean        // 是否已读
}
```

### 3. 界面组件

#### 消息列表组件

```javascript
// 伪代码示例
function MessageList({ messages }) {
  return (
    <div className="message-list">
      {messages.map(msg => (
        <MessageItem key={msg.id} message={msg} />
      ))}

```

#### 输入框组件

```javascript
// 伪代码示例
function ChatInput({ onSend }) {
  const [content, setContent] = useState('');
  const [showEmoji, setShowEmoji] = useState(false);
  
  const handleSend = () => {
    if (content.trim()) {
      onSend(content);
      setContent('');
    }
  };
  
  return (
    <div className="chat-input">
      <input 
        value={content}
        onChange={(e) => setContent(e.target.value)}
        onKeyPress={(e) => e.key === 'Enter' && handleSend()}
      />
      <button onClick={() => setShowEmoji(!showEmoji)}>😊</button>
      <button onClick={handleSend}>发送</button>
      {showEmoji && <EmojiPicker onSelect={setContent} />}
    </div>
  );
}
```

#### 在线状态

- 实时显示用户在线/离线状态

## 用户体验要求

### 1. 界面设计

- 参考 QQ iPad 版的布局设计
- 清晰的消息气泡显示
- 合理的空间利用
- 适配横屏平板

### 2. 交互体验

- 消息发送即时显示
- 滚动加载历史消息
- 新消息自动滚动到底部
