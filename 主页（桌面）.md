# 主页（桌面）系统

## 系统概述

主页是用户登录后的主要操作界面，参考 iPad 桌面设计，提供直观、美观的应用启动和管理入口。

## 界面布局

### 整体结构

```
┌─────────────────────────────────────────────────┐
│              超能岛系统（顶部）                   │
├─────────────────────────────────────────────────┤
│                                                 │
│                                                 │
│              桌面背景区域                        │
│                                                 │
│                                                 │
├─────────────────────────────────────────────────┤
│           Dock 栏（应用入口）                     │
└─────────────────────────────────────────────────┘
```

### 1. 桌面背景

- **位置**: 占据屏幕主要区域
- **功能**:
  - 显示用户自定义或系统预设的壁纸
  - 作为整个桌面的视觉基础
- **技术要求**:
  - 支持常见图片格式（JPG、PNG、GIF）
  - 适配横屏平板分辨率
  - 优化加载性能

### 2. 超能岛系统（顶部）

- **位置**: 屏幕顶部中央区域
- **详细说明**: 参考 [超能岛系统.md](./超能岛系统.md)
- **主要功能**:
  - 显示系统通知和提醒
  - 展示管理系统广播
  - 提供实时信息展示
  - 支持弹性动画效果

### 3. Dock 栏（底部应用入口）

- **位置**: 屏幕底部
- **设计风格**: 参考 iOS Dock 栏设计
- **功能**:
  - 放置常用应用快捷入口
  - 支持应用图标悬浮效果
  - 点击图标打开对应应用

#### Dock 应用入口规划

| 应用    | 图标  | 说明         |
| ----- | --- | ---------- |
| 聊天系统  | 💬  | 进入公共聊天室/私聊 |
| AI 聊天 | 🤖  | AI 智能对话    |
| 天气系统  | 🌤️ | 查看天气信息     |
| 资源仓库  | 📁  | 浏览资源文件     |
| 设置    | ⚙️  | 用户设置和管理中心  |

## 技术实现要点

### 1. 布局设计

```css
/* 伪代码示例 */
.desktop {
  display: grid;
  grid-template-rows: auto 1fr auto; /* 超能岛 - 背景 - Dock */
  height: 100vh;
  width: 100vw;
}
```

### 2. 响应式适配

- 适配横屏平板分辨率（如 1920×1080、2560×1440 等）
- 使用相对单位（rem、vh、vw）进行布局
- 确保在不同尺寸平板上的显示效果

### 3. 动画效果

- Dock 图标悬停放大效果
- 应用打开的过渡动画
- 与超能岛的联动动画

### 4. 应用启动机制

```javascript
// 伪代码示例
function openApp(appId) {
  // 1. 验证用户登录状态
  if (!isAuthenticated()) {
    redirectToLogin();
    return;
  }
  
  // 2. 打开对应应用
  window.location.href = getAppUrl(appId);
}
```

## 用户体验要求

### 1. 视觉设计

- 遵循 iOS 设计语言
- 保持简洁、现代的设计风格
- 使用流畅的过渡动画
- 确保图标和元素的视觉一致性

### 2. 交互设计

- 点击反馈及时
- 应用启动流畅
- 支持快捷操作（如双击、长按等）

### 3. 性能优化

- 桌面背景图片懒加载
- 应用图标预加载
- 减少 DOM 操作，使用 CSS 动画

## 与其他系统的关系

| 关联系统   | 关联方式 | 说明           |
| ------ | ---- | ------------ |
| 超能岛系统  | 组件集成 | 桌面顶部集成超能岛组件  |
| 账户系统   | 认证依赖 | 依赖账户系统进行登录验证 |
| 所有应用系统 | 入口提供 | 作为所有应用的统一入口  |
