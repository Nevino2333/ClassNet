# AI 聊天系统

## 系统概述

AI 聊天系统是 ClassNet 的智能对话模块，支持与开发者提供的 AI 模型（API）进行文本对话，支持人设预设、历史对话等功能，并支持 LaTeX 公式和 Markdown 渲染。

## 核心功能

### 1. AI 对话

#### 功能描述
用户可与 AI 模型进行自然语言对话，获取智能回复。

#### 对话特性
- 支持多轮对话（上下文理解）
- 支持流式输出（打字机效果）
- 支持对话历史保存
- 支持新建对话（清空上下文）

### 4. LaTeX 公式渲染

#### 功能描述
支持在对话中显示和渲染 LaTeX 数学公式。

#### 支持格式
- 行内公式：`$E = mc^2$`
- 块级公式：`$$\int_a^b f(x)dx$$`
- 多行公式：使用 `align` 环境
- 矩阵、分数、根号等常见数学符号

#### 渲染示例
```
行内公式：质能方程 $E = mc^2$ 描述了质量和能量的关系。

块级公式：
$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$

多行公式：
$$
\begin{aligned}
F &= ma \\
E &= mc^2 \\
P &= VI
\end{aligned}
$$
```

#### 技术实现
```javascript
// 使用 MathJax 渲染
import MathJax from 'mathjax';

// 配置 MathJax
MathJax.typesetPromise([element]).then(() => {
  // 渲染完成
});
```

### 5. Markdown 渲染

#### 功能描述
支持 Markdown 格式的文本渲染，提升阅读体验。

#### 支持语法
- 标题（# H1, ## H2, ...）
- 粗体（**text**）
- 斜体（*text*）
- 列表（有序/无序）
- 引用（> quote）
- 代码块（```code```）
- 链接（[text](url)）
- 表格

#### 渲染示例
```markdown
# 标题

这是一段**粗体**和*斜体*文本。

## 列表
- 项目 1
- 项目 2
- 项目 3

## 引用
> 这是一段引用文本

## 代码
```python
def hello():
    print("Hello, World!")
```
```

#### 技术实现
```javascript
// 使用 marked 或 markdown-it 解析
import { marked } from 'marked';

function renderMarkdown(content) {
  const html = marked.parse(content);
  return html;
}
```

## 技术实现要点

### 1. API 调用

#### 服务端中转（必须）
根据开发规范，所有 API 调用必须经过服务器中转。

```javascript
// 服务端 AI API 调用示例
const axios = require('axios');

const AI_API_KEY = process.env.AI_API_KEY;
const AI_API_URL = process.env.AI_API_URL;

async function chatWithAI(messages, options = {}) {
  const response = await axios.post(AI_API_URL, {
    model: options.model || 'default',
    messages: messages,
    stream: options.stream || false,
    temperature: options.temperature || 0.7,
    max_tokens: options.maxTokens || 2000
  }, {
    headers: {
      'Authorization': `Bearer ${AI_API_KEY}`,
      'Content-Type': 'application/json'
    }
  });
  
  return response.data;
}

// 消息格式
const messages = [
  {
    role: 'system',
    content: systemPrompt  // 人设预设
  },
  {
    role: 'user',
    content: '用户的问题'
  }
];
```

### 2. 流式输出

#### WebSocket 流式传输
```javascript
// 服务端流式响应
async function streamChat(messages, ws) {
  const response = await fetch(AI_API_URL, {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${AI_API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      messages,
      stream: true
    })
  });
  
  const reader = response.body.getReader();
  
  while (true) {
    const { done, value } = await reader.read();
    if (done) break;
    
    const chunk = new TextDecoder().decode(value);
    ws.send(JSON.stringify({
      type: 'chunk',
      content: chunk
    }));
  }
  
  ws.send(JSON.stringify({ type: 'end' }));
}
```

#### 前端流式显示
```javascript
// 前端接收流式数据
ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  
  if (data.type === 'chunk') {
    appendToMessage(data.content);  // 打字机效果
    renderMarkdownAndLatex();       // 实时渲染
  } else if (data.type === 'end') {
    saveConversation();             // 保存对话
  }
};
```
